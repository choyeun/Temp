JavaScript - alert

  • 1. alert() 함수는 경고창을 띄웁니다.
  • 2. 경고창에 표시할 메시지를 인자로 전달합니다.
  • 3. 경고창에 표시할 메시지는 문자열로 전달합니다.
  • 4. 경고창에 표시할 메시지는 큰 따옴표("")로 감싸야 합니다.

alert 실행 전 코드

<script> alert("Step-1 : alert"); </script>

alert 실행후 반환 값

  • 1. alert() 함수는 경고창을 띄우고 사용자가 확인 버튼을 누르면 undefined를 반환합니다.
  • 2. 사용자가 확인 버튼을 누르기 전까지는 다음 코드로 넘어가지 않습니다.
