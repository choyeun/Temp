JavaScript - if else

  • 1. if else 문은 조건문입니다.
  • 2. 조건문은 조건식을 평가하여 참 또는 거짓을 판단합니다.
  • 3. 조건식이 참이면 if 블록을 실행하고, 거짓이면 else 블록을 실행합니다.
  • 4. else 블록은 생략할 수 있습니다.

if else 실행 전 코드

<script> var num = 3; if (num > 5) { document.write("num은 5보다 큽니다."); } else if (num === 5) { document.write("num은 5입니다."); } else { document.write("num은 5보다 작습니다."); } </script>
