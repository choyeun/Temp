<iFRAME> … </iFRAME>

iFRAME tag 는 중첩 브라우징을 나타내는 요소로, 현재 문서 안에 다른 HTML 페이지를 삽입합니다.

예시 사이트

<h3>&lt;iFRAME&gt; … &lt;/iFRAME&gt;</h3> <hr> iFRAME tag 는 중첩 브라우징을 나타내는 요소로, 현재 문서 안에 다른 HTML 페이지를 삽입합니다. <hr> <h3>예시 사이트</h3> <hr> <iframe src="https://www.anyang.ac.kr/main/index.do" name="upper" width="1000" height="1000"></iframe>
