JavaScript - 텍스트 그리기(내 이름)

  • 1. HTML5 canvas 요소를 사용하여 텍스트를 그립니다.
  • 2. canvas 요소는 그래픽을 그리기 위한 요소입니다.
  • 3. getContext() 메서드로 그리기 컨텍스트를 얻습니다.
  • 4. fillText() 메서드로 텍스트를 그립니다.

텍스트 그리기 실행 전 코드

<canvas id="myCanvas" width="200" height="100" style="border:1px solid #000000;"></canvas> <script> var canvas = document.getElementById("myCanvas"); var ctx = canvas.getContext("2d"); ctx.font = "30px Arial"; ctx.fillText("김인찬", 10, 50); </script>
