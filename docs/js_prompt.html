JavaScript - prompt

  • 1. prompt() 함수는 입력창을 띄웁니다.
  • 2. 입력창에 표시할 메시지를 인자로 전달합니다.
  • 3. 입력창에 표시할 메시지는 문자열로 전달합니다.
  • 4. 입력창에 표시할 메시지는 큰 따옴표("")로 감싸야 합니다.

prompt 실행 전 코드

<script> var result = prompt("Step-1 : prompt"); document.write("Step-2 : " + result); </script>

prompt 실행후 반환 값

  • 1. prompt() 함수는 입력창을 띄우고 사용자가 입력한 값을 반환합니다.
  • 2. 사용자가 확인 버튼을 누르기 전까지는 다음 코드로 넘어가지 않습니다.
