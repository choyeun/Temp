JavaScript - switch case

  • 1. switch case 문은 다중 조건문입니다.
  • 2. switch case 문은 조건식을 평가하여 일치하는 case 문을 실행합니다.
  • 3. case 문이 일치하지 않으면 default 문을 실행합니다.
  • 4. default 문은 생략할 수 있습니다.

switch case 실행 전 코드

<script> var num = 5; switch (num) { case 1: document.write("num은 1입니다."); break; case 2: document.write("num은 2입니다."); break; case 3: document.write("num은 3입니다."); break; default: document.write("num은 1, 2, 3이 아닙니다."); } </script>

switch case 실행후 반환 값

  • 1. switch case 문은 조건식을 평가하여 일치하는 case 문을 실행합니다.
  • 2. case 문이 일치하지 않으면 default 문을 실행합니다.
  • 3. default 문은 생략할 수 있습니다.
