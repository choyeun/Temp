JavaScript - 구구단

  • 1. 구구단은 1단부터 9단까지 출력합니다.
  • 2. for 문을 중첩하여 구구단을 출력합니다.
  • 3. for 문을 중첩할 때는 외부 for 문과 내부 for 문을 구분합니다.
  • 4. 외부 for 문은 단을, 내부 for 문은 곱을 출력합니다.

구구단 실행 전 코드

<script> for (var i = 1; i <= 9; i++) { document.write("<h3>" + i + "단</h3>"); for (var j = 1; j <= 9; j++) { document.write(i + " * " + j + " = " + i * j + "<br>"); } } </script>
