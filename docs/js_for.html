JavaScript - for

  • 1. for 문은 반복문입니다.
  • 2. for 문은 특정 횟수만큼 코드를 반복 실행합니다.
  • 3. for 문은 초기식, 조건식, 증감식을 가지고 있습니다.
  • 4. 초기식은 반복문이 시작할 때 한 번만 실행됩니다.
  • 5. 조건식은 반복문이 실행될 조건을 지정합니다.
  • 6. 증감식은 반복문이 실행된 후 실행됩니다.

for 실행 전 코드

<script> for (var i = 0; i < 5; i++) { document.write("i = " + i + "<br>"); } </script>
