JavaScript - while

  • 1. while 문은 반복문입니다.
  • 2. 조건식이 참인 동안 코드 블록을 반복 실행합니다.
  • 3. 조건식이 거짓이면 코드 블록을 실행하지 않습니다.
  • 4. 조건식이 참인 동안 코드 블록을 반복 실행합니다.

while 실행 전 코드

<script> var num = 1; while (num <= 5) { document.write("num은 " + num + "입니다.<br>"); num++; } </script>
