JavaScript - confirm

  • 1. confirm() 함수는 확인창을 띄웁니다.
  • 2. 확인창에 표시할 메시지를 인자로 전달합니다.
  • 3. 확인창에 표시할 메시지는 문자열로 전달합니다.
  • 4. 확인창에 표시할 메시지는 큰 따옴표("")로 감싸야 합니다.

confirm 실행 전 코드

<script> var result = confirm("Step-1 : confirm"); document.write("Step-2 : " + result); </script>

confirm 실행후 반환 값

  • 1. confirm() 함수는 확인창을 띄우고 사용자가 확인 버튼을 누르면 true를 반환합니다.
  • 2. confirm() 함수는 확인창을 띄우고 사용자가 취소 버튼을 누르면 false를 반환합니다.
  • 3. 사용자가 확인 버튼을 누르기 전까지는 다음 코드로 넘어가지 않습니다.
