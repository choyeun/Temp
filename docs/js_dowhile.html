JavaScript - do while

  • 1. do while 문은 반복문입니다.
  • 2. do while 문은 조건식을 평가하여 참이면 블록을 실행하고, 거짓이면 블록을 종료합니다.
  • 3. do while 문은 블록을 먼저 실행하고 조건식을 평가합니다.
  • 4. do while 문은 블록을 최소 한 번 실행합니다.

do while 실행 전 코드

<script> var num = 1; do { document.write("num은 " + num + "입니다.<br>"); num++; } while (num <= 3); </script>
